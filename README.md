# coding problems with solutions from beecrowd

This repository contains solutions 200 + problems from BeeCrowd, solved using C++.

## Introduction

[BeeCrowd](https://www.beecrowd.com.br/) is a Brazilian platform for competitive programming enthusiasts. It hosts a variety of problems ranging from beginner to advanced levels, providing an excellent resource for honing problem-solving skills.

## Contents

The repository is organized into folders, each containing solutions to problems from different categories or difficulty levels. Here's a brief overview:

- **Beginner Problems**: Solutions to basic problems suitable for newcomers to competitive programming.
- **Intermediate Problems**: Solutions to problems of moderate difficulty, suitable for those with some programming experience.
- **Advanced Problems**: Solutions to more challenging problems for experienced programmers looking to refine their skills.

## Getting Started

To explore the solutions or contribute to this repository, follow these steps:

1. Clone the repository to your local machine:

   ```bash
   git clone https://github.com/kawsarkabir/BeeCrowd-Cpp.git
   ```

2. Navigate to the desired category/folder containing the problem solutions.
3. Open the solution files using a C++ compiler or your preferred Integrated Development Environment (IDE).
4. Explore the solutions, try running them, and analyze the code to understand the problem-solving techniques employed.

## Contribution Guidelines

Contributions to this repository are welcome! If you'd like to contribute solutions to additional problems or improve existing ones, please follow these guidelines:

1. Fork the repository to your GitHub account.
2. Create a new branch for your changes:

   ```bash
   git checkout -b feature/new-solutions
   ```

3. Add your solutions or make improvements to existing ones.
4. Test your changes to ensure they work correctly.
5. Commit your changes with descriptive commit messages:

   ```bash
   git commit -m "Add solution to problem ABC"
   ```

6. Push your changes to your fork:

   ```bash
   git push origin feature/new-solutions
   ```

7. Open a pull request from your fork to the main repository, explaining your changes.

## Disclaimer

Please note that while this repository provides solutions to BeeCrowd problems, it's essential to approach competitive programming with a focus on understanding problem-solving techniques rather than just copying solutions. Use these solutions as a reference to learn and improve your skills.
